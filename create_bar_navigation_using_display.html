Website Navigation

What is it?

Website navigation is one of the major factors for websites, they have a primary responsiblility and purpose for moving their visitors from one page to another. Without a navigation, it would be difficult for website visitors to adapt the interest on browsing with and becomes more frustrating for them to find pages they want to visit.

How does it work?

When websites are visited, visitors often see a navigation across every page in different ways and approaches to directing them to a specific page. Those may be seen to be Bar Navigation, Side Navigation and Footer Navigation. They all have different purposes on how they are used to interact for the visitor.

Bar Navigation

In this topic, will explain about creating a Repsonsive Bar Navigation with the display method. Constucting a navigation has to be planned out well before building, we have to consider why and how to want our navigation to interact. Most importantly, we need to consider how we would design to attract interest to our target audiences.

Requirements
  1. HTML
  2. CSS

Create the Navigation

Create a html file where we will create our bar navigation on.

HTML Nav Bar
<div id="nav-bar">   <a href="#home">Home</a>   <a href="#page1">Page 1</a>   <a href="#page2">Page 2</a>   <a href="#page3">Page 3</a> </div>

Creating a div container called nav-bar as our id containing each link inside with the a tags.

Don't Float Just Display

One of the main differences why we could use Display instead of Float is that we can align our navigation responsive as possible. We can neatly position the navigation without it spreading across the page. Float are diffcult to align centering but can only make it responsive using percentage for most cases but shrinking can cause navigation to be difficult scale. Using display can make a nvigation more responsive as it can get. Some considerations are in place to mind.

Styling the Nav Bar with CSS
#nav-bar {   background-color: MediumSpringGreen;   width: 100%;   height: 50px;   text-align: center; }
#nav-bar a {   background-color: teal;   color: white;   padding: 16px 10px;   font-size: 16px;   font-weight: bold;   text-decoration: none; }

Inline and Inline-Block

Display method has different elements of how they are used for navigation methods. Three elements you should know about are inline, Inline-Block and Block elements.

Adding Inline-Block
#nav-bar a {   background-color: teal;   color: white;   padding: 16px 10px;   font-size: 16px;   font-weight: bold;   text-decoration: none;   /* Add the display element of inline-block */   display: inline-block; }

Resolving Problem with Inline Navigation Gaps and Overlaps

Display method has different elements of how they are used for navigation methods. Three elements you should know about are inline, Inline-Block and Block elements.

Fixing Gaps and Overlaps
#nav-bar a {   background-color: teal;   color: white;   padding: 16px 10px;   font-size: 16px;   font-weight: bold;   text-decoration: none;   display: inline-block;   /* Add the margin for the navigation links */   margin: 0 -2px 0 -2.5px;   /* Sometimes you might need to adjust the right margins */ }
